## 🎯 Features

- 🌳 **Tree Structure Management** - Hierarchical prompt organization with visual drag-and-drop editing
- 🔗 **Dependency Configuration** - Configure node dependencies with automatic runtime concatenation
- ▶️ **Node-level Execution** - Run buttons, loading states, and callback handling
- 🤖 **AI Optimization** - Intelligent prompt content optimization
- 🔒 **Lock Mechanism** - Lock after execution to prevent accidental changes
- ⚡ **High Performance** - Map data structure supporting 2000+ nodes
- 📝 **CodeMirror Integration** - Markdown syntax highlighting with lazy loading
- 🎨 **Customizable Theme** - CSS variables with theme extension support
- 🌍 **Internationalization** - Multi-language support (Chinese & English)
