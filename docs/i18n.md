# 国际化 (i18n)

React Prompt Editor 支持国际化，默认语言为中文。

## 使用方式

### 1. 导入语言包

类似 Ant Design 的使用方式：

```typescript
import { PromptEditor, zhCN, enUS } from 'react-prompt-editor';
```

### 2. 配置语言

在 `PromptEditor` 组件中通过 `locale` 属性传入语言包：

```typescript
import { PromptEditor, zhCN, enUS } from 'react-prompt-editor';

// 使用中文（默认）
const App = () => (
  <PromptEditor
    locale={zhCN}
    // ...其他 props
  />
);

// 使用英文
const AppEn = () => (
  <PromptEditor
    locale={enUS}
    // ...其他 props
  />
);
```

### 3. 自定义翻译函数（高级用法）

如果需要更灵活的翻译控制，可以使用 `createTranslator`：

```typescript
import { createTranslator, zhCN } from 'react-prompt-editor';

const t = createTranslator(zhCN);

// 使用翻译
console.log(t('editor.addRootNode')); // "添加一级标题"
console.log(t('optimize.selectedContent', { length: 100 })); // "已选中 100 字"
```

## 支持的语言

- **zhCN**: 简体中文（默认）
- **enUS**: 英文

## CodeMirror 编辑器国际化

CodeMirror 编辑器的搜索框等 UI 元素也会根据 `locale` 配置自动切换语言。

完整示例请参考：[国际化演示](./components/prompt-editor#🌍-国际化)

支持的 CodeMirror 翻译包括：

- 搜索/替换文本
- 区分大小写/全字匹配/正则表达式
- 上一个/下一个匹配
- 关闭按钮等

## 扩展语言包

如需添加其他语言，可以参考 `src/i18n/locales/zh-CN.ts` 的结构创建新的语言包：

```typescript
// src/i18n/locales/ja-JP.ts
export default {
  common: {
    confirm: '確認',
    cancel: 'キャンセル',
    // ...
  },
  editor: {
    addRootNode: 'ルートタイトルを追加',
    // ...
  },
  // ...
};
```

## 语言包结构

语言包包含以下模块：

- `common`: 通用文本
- `editor`: 编辑器相关文本
- `node`: 节点操作文本
- `dependency`: 依赖配置文本
- `optimize`: AI 优化弹窗文本
- `editor_toolbar`: 编辑器工具栏文本
- `message`: 消息提示文本

## 注意事项

1. 如果不传 `locale` 属性，组件将使用默认的中文语言包
2. 语言包是纯对象，可以直接导入使用，无需额外配置
3. 所有文本都支持变量替换，使用 `{variable}` 格式
