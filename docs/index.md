## 🎯 特性

- 🌳 **树形结构管理** - 层级化提示词组织，可视化拖拽编辑
- 🔗 **依赖关系配置** - 配置节点间依赖，运行时自动拼接
- ▶️ **节点级运行** - 运行按钮、Loading 状态、回调处理
- 🤖 **AI 优化** - 智能优化提示词内容
- 🔒 **锁定机制** - 运行后锁定，防止误改
- ⚡ **高性能** - Map 数据结构，支持 2000+ 节点
- 📝 **CodeMirror** - Markdown 语法高亮，懒加载
- 🎨 **可定制主题** - CSS 变量，支持主题扩展
