# Internationalization (i18n)

React Prompt Editor supports internationalization with Chinese as the default language.

## Usage

### 1. Import Language Packs

Similar to Ant Design's approach:

```typescript
import { PromptEditor, zhCN, enUS } from 'react-prompt-editor';
```

### 2. Configure Language

Pass the language pack through the `locale` prop in the `PromptEditor` component:

```tsx
import { PromptEditor, zhCN, enUS } from 'react-prompt-editor';

// Use Chinese (default)
const App = () => (
  <PromptEditor
    locale={zhCN}
    // ...other props
  />
);

// Use English
const AppEn = () => (
  <PromptEditor
    locale={enUS}
    // ...other props
  />
);
```

### 3. Custom Translation Function (Advanced)

For more flexible translation control, you can use `createTranslator`:

```typescript
import { createTranslator, zhCN } from 'react-prompt-editor';

const t = createTranslator(zhCN);

// Use translation
console.log(t('editor.addRootNode')); // "添加一级标题"
console.log(t('optimize.selectedContent', { length: 100 })); // "已选中 100 字"
```

## Supported Languages

- **zhCN**: Simplified Chinese (default)
- **enUS**: English

## CodeMirror Editor Internationalization

CodeMirror editor UI elements like search boxes will automatically switch languages based on the `locale` configuration:

```tsx
import { PromptEditor, zhCN, enUS } from 'react-prompt-editor';

// CodeMirror search box will display Chinese
const App = () => <PromptEditor locale={zhCN} />;

// CodeMirror search box will display English
const AppEn = () => <PromptEditor locale={enUS} />;
```

Supported CodeMirror translations include:

- Search/replace text
- Case sensitive/whole word match/regular expression
- Previous/next match
- Close button, etc.

## Extending Language Packs

To add other languages, you can follow the structure in `src/i18n/locales/zh-CN.ts` to create new language packs:

```typescript
// src/i18n/locales/ja-JP.ts
export default {
  common: {
    confirm: '確認',
    cancel: 'キャンセル',
    // ...
  },
  editor: {
    addRootNode: 'ルートタイトルを追加',
    // ...
  },
  // ...
};
```

## Language Pack Structure

Language packs contain the following modules:

- `common`: Common text
- `editor`: Editor-related text
- `node`: Node operation text
- `dependency`: Dependency configuration text
- `optimize`: AI optimization dialog text
- `editor_toolbar`: Editor toolbar text
- `message`: Message notification text

## Notes

1. If the `locale` prop is not provided, the component will use the default Chinese language pack
2. Language packs are plain objects that can be imported and used directly without additional configuration
3. All text supports variable substitution using `{variable}` format
