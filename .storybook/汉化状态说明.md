1. **官方语言包支持** - Storybook 官方正在开发完整的 i18n 系统

3. **Preview 端配置** - 需要配置 preview 端的翻译

### 🔧 已完成的配置

#### 1. 安装 storybook-i18n
```bash
npm install -D storybook-i18n@10
```

#### 2. 配置 main.ts
```typescript
{
  addons: [
    // ... 其他 addons
    "storybook-i18n"
  ]
}
```

#### 3. 配置 preview.tsx
```typescript
const preview: Preview = {
  initialGlobals: {
    locale: 'zh-CN',
    locales: {
      'zh-CN': { icon: '🇨🇳', title: '简体中文', right: 'CN' },
      'en': { icon: '🇺🇸', title: 'English', right: 'US' },
    },
  },
};
```

#### 4. 配置 manager.ts
```typescript
import { addons } from 'storybook/manager-api'

addons.setConfig({
  initialGlobals: {
    locale: 'zh-CN',
  },
})
```

## 📝 编写中文文档

虽然界面可能部分显示英文，但你可以在故事文件中完全使用中文：

### 示例：Button.stories.ts

```typescript
import type { Meta, StoryObj } from '@storybook/react-vite'
import { Button } from './Button'

const meta = {
  title: '示例/按钮',  // ✅ 中文路径
  component: Button,
  parameters: {
    docs: {
      description: {
        component: `
## Button 按钮

按钮是常用的基础组件，用于触发操作或跳转链接。

### 使用场景
- 表单提交
- 对话框操作
- 导航跳转
        `,  // ✅ 中文描述
      },
    },
  },
  argTypes: {
    label: {
      description: '按钮文本',  // ✅ 中文参数说明
    },
    primary: {
      description: '是否为主要样式',
    },
  },
} satisfies Meta<typeof Button>

export default meta
type Story = StoryObj<typeof meta>

export const 主要按钮：Story = {
  args: {
    primary: true,
    label: '提交',  // ✅ 中文参数值
  },
}
```

## 💡 替代方案

如果需要完全的界面汉化，可以考虑：

### 方案 1：使用 react-i18next
适用于 React 项目，可以完全控制翻译。

```bash
npm install -D storybook-react-i18next
npm install i18next react-i18next i18next-browser-languagedetector i18next-http-backend
```

### 方案 2：使用 react-intl
适用于使用 FormatJS 的项目。

```bash
npm install -D storybook-react-intl
npm install react-intl
```

### 方案 3：自定义 i18n 装饰器
创建项目特定的语言切换装饰器。

```typescript
// .storybook/preview.tsx
import { useGlobals } from '@storybook/manager-api'

const withI18n = (Story, context) => {
  const [{ locale }] = useGlobals()
  // 根据 locale 应用翻译
  return <Story />
}

const preview: Preview = {
  decorators: [withI18n],
  initialGlobals: {
    locale: 'zh-CN',
    locales: {
      'zh-CN': '简体中文',
      'en': 'English',
    },
  },
}
```

## ❓ 常见问题

### Q: 为什么切换语言后界面还是英文？
A: Storybook 10.x 的核心界面文本汉化需要官方语言包支持。目前 `storybook-i18n` 主要提供语言切换框架，完整的翻译需要 Storybook 官方的支持。

### Q: 如何查看当前支持的翻译？
A: 查看 `.storybook/i18n.ts` 文件中定义的翻译键值对。

### Q: 可以自定义翻译吗？
A: 可以！编辑 `.storybook/i18n.ts` 文件，添加你的翻译。

### Q: 文档区域会跟随语言切换吗？
A: 会的！在 `preview.tsx` 中配置的 `initialGlobals.locale` 会影响文档区域的语言。
