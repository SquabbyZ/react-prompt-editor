# Storybook 新手入门指南

## 🎯 什么是 Storybook？

Storybook 是一个**前端组件开发工具**，可以帮助你：

- ✅ **独立开发组件** - 不需要运行整个应用
- ✅ **可视化测试** - 直观地查看组件的各种状态
- ✅ **自动生成文档** - 组件 API 文档自动生成
- ✅ **团队协作** - 设计师和产品经理可以查看组件库

## 📖 核心概念

### 什么是"故事"（Story）？

**故事** = 组件的**一种状态**或**一种用法**

例如 Button 组件：
- 主要按钮是一个故事
- 次要按钮是另一个故事
- 禁用状态又是一个故事

### 故事文件结构

```typescript
// Button.zh-CN.stories.ts

// 1. 导入组件和类型
import type { Meta, StoryObj } from '@storybook/react-vite'
import { Button } from './Button'

// 2. 配置元数据
const meta = {
  title: '示例/按钮',        // 在侧边栏显示的路径
  component: Button,         // 组件
  parameters: {
    layout: 'centered',      // 布局方式
    docs: {
      description: {
        component: '组件说明文档...'  // 组件描述
      },
    },
  },
  argTypes: {
    // 控件配置
    label: {
      description: '按钮文本',
      control: 'text',
    },
  },
} satisfies Meta<typeof Button>

export default meta
type Story = StoryObj<typeof meta>

// 3. 定义具体的故事
export const 主要按钮：Story = {
  args: {
    primary: true,
    label: '主要按钮',
  },
}
```

## 🎨 使用控件面板

在 Storybook 界面底部，你会看到**控件**面板：

- 🎛️ **实时调整属性** - 修改按钮文本、颜色、尺寸等
- 👁️ **即时预览** - 修改后立即看到效果
- 🔄 **重置** - 一键恢复到默认值

## 📝 编写你的第一个故事

### 步骤 1：创建组件

在 `src/components/` 创建你的组件：

```tsx
// src/components/Hello.tsx
interface HelloProps {
  name: string
}

export function Hello({ name }: HelloProps) {
  return <h1>你好，{name}！</h1>
}
```

### 步骤 2：创建故事文件

在 `src/stories/` 创建故事：

```typescript
// src/stories/Hello.stories.ts
import type { Meta, StoryObj } from '@storybook/react-vite'
import { Hello } from '../components/Hello'

const meta = {
  title: '我的组件/Hello',
  component: Hello,
  parameters: {
    layout: 'centered',
  },
  args: {
    name: '世界',
  },
} satisfies Meta<typeof Hello>

export default meta
type Story = StoryObj<typeof meta>

export const 默认：Story = {}

export const 自定义名字：Story = {
  args: {
    name: 'Storybook',
  },
}
```

### 步骤 3：查看效果

保存文件后，Storybook 会自动刷新，在侧边栏找到你的组件！

## 💡 最佳实践

### ✅ 推荐

- **为每种状态创建故事** - 正常、禁用、加载等
- **使用有意义的故事名** - 如 `主要按钮`、`禁用状态`
- **添加详细的文档** - 在 `args` 中描述每个属性
- **使用控件测试** - 确保所有属性都能正常工作

### ❌ 避免

- **不要在故事中写复杂逻辑**
- **不要依赖外部状态**
- **不要忘记无障碍测试**

## ❓ 常见问题

### Q: 为什么我的组件没有显示？
A: 检查：
- 故事文件是否在 `src/stories/` 目录
- 文件名是否以 `.stories.ts` 或 `.stories.tsx` 结尾
- 是否正确导出了 `meta` 和故事

### Q: 如何自定义主题？
A: 在 `.storybook/preview.tsx` 中配置 `parameters.theme`

### Q: 如何集成到我的项目？
A: Storybook 是独立的开发环境，不影响你的主应用。开发完成后，将组件复制到你的项目中即可。
